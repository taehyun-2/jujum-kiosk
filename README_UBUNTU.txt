우분투 실행 방법

1. 압축 풀기
2. 압축 푼 폴더에서 터미널 열기
3. 실행

chmod +x start_ubuntu.sh
./start_ubuntu.sh

접속 주소

관리자:
http://localhost:3000/admin

테이블:
http://localhost:3000/table/1
...
http://localhost:3000/table/70

이번 수정 내용

- 누적금액은 테이블 전체 기준이 아니라 손님 폰 브라우저 기준입니다.
- 같은 테이블 QR을 다른 사람이 찍어도 이전 손님 주문 내역은 보이지 않습니다.
- 새 주문번호는 계속 따로 생성됩니다.
- 손님이 같은 폰에서 추가 주문하면 자기 주문 누적금액만 표시됩니다.
- 취소 주문은 손님 누적금액에서 제외됩니다.
- 메뉴 담기 버튼을 누르면 버튼이 잠깐 연해집니다. 별도 '담김' 문구는 뜨지 않습니다.
